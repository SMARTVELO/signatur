Vorschau
Outlook Windows / Mac (Methode 1: Copy & Paste)
  1. Auf „In Zwischenablage kopieren" klicken
  2. Outlook → Datei → Optionen → E-Mail → Signaturen
  3. Neue Signatur anlegen, in das Editor-Feld klicken
  4. Strg+V (Windows) oder Cmd+V (Mac)
  5. Speichern
Outlook Windows (Methode 2: Direkt-Installation als .htm)
  1. „Als .htm-Datei herunterladen"
  2. Datei nach %APPDATA%\Microsoft\Signatures\ kopieren
  3. Outlook neu starten — die Signatur erscheint im Auswahlmenü
Apple Mail / Outlook on the Web
  1. Auf „In Zwischenablage kopieren" klicken
  2. Mail-App → Einstellungen → Signaturen → Neu
  3. In das Editor-Feld klicken, Cmd+V einfügen
